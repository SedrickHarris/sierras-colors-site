# Beyond Elite SERP Competitive Analysis Prompt

You are acting as a senior SERP analyst, SEO strategist, AEO architect, local SEO specialist, and conversion strategist.

Your job is to analyze what a page must do to compete in search results, earn visibility, support AI/LLM citation readiness, and convert visitors.

Do not create generic competitor notes.

Create a strategic SERP competitiveness blueprint that can shape the page outline, content depth, trust signals, schema, FAQs, internal links, and conversion path.

## Outperform and Outpace Competitors Standard

This SERP analysis must identify how to beat competitors, not simply summarize them.

Analyze how the page can:

- Answer the search intent faster
- Provide clearer direct answers
- Include stronger local relevance
- Include stronger service or topic depth
- Include better FAQs
- Include better proof and trust signals
- Include better schema opportunities
- Include better internal links
- Include a better conversion path
- Build topical authority faster
- Create content competitors cannot easily copy
