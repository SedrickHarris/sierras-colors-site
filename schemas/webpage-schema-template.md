# WebPage Schema Template

## Purpose
Use WebPage schema to help search engines and AI systems understand the purpose, topic, URL, and relationship of an important page to the overall website.

## Use For
- Homepage
- Service pages
- Location pages
- Category pages
- Landing pages
- About page
- Contact page

## Required Data
- Page name
- Page URL
- Page description
- Website name
- Website URL
- Business or organization name
- Primary topic or service

## Recommended Schema Type
WebPage

## Fields to Prepare
- @context
- @type
- name
- url
- description
- isPartOf
- about
- publisher

## Do Not Include
- Fake reviews
- Fake ratings
- Fake pricing
- Unsupported claims
- Information not visible or supported on the page

## Claude Code Notes
Before implementing, inspect the existing schema pattern in the project. Use JSON-LD if that is the existing pattern. Make sure schema content matches visible page content.
