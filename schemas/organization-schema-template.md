# Organization Schema Template

## Purpose
Use Organization schema to define the business entity, brand name, website, logo, contact points, and official profiles.

## Use For
- Homepage
- About page
- Global site schema
- Footer schema
- Brand and entity authority support

## Required Data
- Business name
- Website URL
- Logo URL if available
- Business description
- Public contact information if available
- Official social or profile URLs if available

## Recommended Schema Type
Organization

## Fields to Prepare
- @context
- @type
- name
- url
- logo
- description
- sameAs
- contactPoint

## Do Not Include
- Social links that are not official
- Fake founding dates
- Fake awards
- Fake credentials
- Unsupported claims

## Claude Code Notes
Use Organization schema when the site needs stronger brand and entity clarity. If the business is local, LocalBusiness schema may also be needed.
