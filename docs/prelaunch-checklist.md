# Prelaunch Checklist

- Confirm verified business name, phone, email, address or service area details.
- Confirm final navigation.
- Confirm launch page copy.
- Confirm service pages and location pages.
- Confirm metadata for each route.
- Confirm schema for each page type.
- Confirm sitemap routes.
- Confirm robots file if needed.
- Confirm Google Search Console setup.
- Confirm Bing Webmaster setup.
- Confirm analytics setup.
- Confirm form and workflow integrations.
- Confirm review automation plan.
- Run `npm run build` before launch.
