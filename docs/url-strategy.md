# URL Strategy

## URL rules

- Use lowercase URLs.
- Use descriptive keyword-focused slugs.
- Use trailing slashes.
- Avoid dates in evergreen service URLs.
- Avoid vague slugs when a service or location keyword is available.

## Service URL structure

`/service-name/`

Examples:

- `/residential-painting/`
- `/commercial-painting/`
- `/interior-painting/`

## Location URL structure

`/service-areas/location-nv/`

Examples:

- `/service-areas/henderson-nv/`
- `/service-areas/las-vegas-nv/`

## Service plus location URL structure

`/service-name/location-nv/`

Examples:

- `/exterior-painting/henderson-nv/`
- `/commercial-painting/las-vegas-nv/`

## Audience plus service URL structure

Use only when planned and approved.

`/audience-service-name/`

Examples:

- `/hoa-painting/`
- `/property-management-painting/`
- `/realtor-painting-services/`

## Blog URL structure

`/blog/topic-slug/`

Blog posts should support service pages, location pages, FAQs, and answer focused search opportunities.
